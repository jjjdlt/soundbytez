# Soundbytez  
*A Web-Based Digital Audio Workstation (DAW) powered by [Tone.js](https://tonejs.github.io/).*

---

## Overview

**Soundbytez** is a browser-based **digital audio workstation (DAW)** designed to provide an accessible, lightweight platform for creating, editing, and playing music directly in the web browser. Built on **Tone.js**, it leverages modern web audio APIs for real-time synthesis, sequencing, and effect processing ‚Äî no installations required.

## Installation & Setup

Clone the repository and install dependencies:

```bash
git clone https://github.com/jjjdlt/soundbytez.git
cd soundbytez
npm install
npm run start
```
Then open your browser and navigate to:

http://localhost:3000
